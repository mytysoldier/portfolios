# 🎙 1min Voice Journal

> **1分の声で、今日の気持ちを書き残す。**  
> AIが感情を分析し、要約とアドバイスを返してくれるボイス日記アプリ。

---

## ✨ Overview

**1min Voice Journal** は、話すだけで気持ちを記録できるアプリです。  
文章を書く手間なく続けられる日記習慣を実現し、AIのサポートでメンタルの変化に気づきやすくします。

- 🎙 **音声で記録**
- 🧠 **AIが感情分析・要約・アドバイス生成**
- 💾 **データは端末にローカル保存（プライバシー配慮）**
- 📅 **日毎に振り返り**

---

## 🎯 Target Users

- 日記が続かない人
- 感情の波を振り返りたい人
- テキストを書くのが苦手な人
- 仕事・ランニング・生活の感情ログを残したい人

---

## 🧱 MVP Features

### 🌟 実装済み / 実装予定の範囲
| 機能 | 内容 |
|------|------|
| 音声録音 | 1分以内、録音/停止ボタン |
| 音声保存 | `Documents/Audio/xxxx.m4a` に保存 |
| 文字起こし | Whisper API |
| AI感情分析 | GPT で JSON 生成（Emotion / Title / Summary / Advice） |
| ローカル保存 | SwiftData で永続化 |
| 一覧表示 | 日付 / 感情アイコン / タイトル |
| 詳細画面 | 感情 / テキスト / 再生機能 |

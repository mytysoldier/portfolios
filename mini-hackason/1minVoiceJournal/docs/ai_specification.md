# 🤖 AI仕様 (AI Specifications)

このアプリケーションでは、ユーザーの音声ジャーナルを分析し、より深い洞察を提供するために、以下のAI機能が利用されています。

## 1. 音声認識 (Speech-to-Text)

- **役割 (Role)**: ユーザーが録音した音声をテキストデータに変換します。
- **機能 (Functionality)**: 録音完了後、自動的に音声が文字起こしされ、次の感情分析プロセスに利用されます。

## 2. 感情分析と要約 (Emotion Analysis & Summarization)

- **役割 (Role)**: 文字起こしされたテキストからユーザーの感情を特定し、ジャーナルの主要な内容を要約し、パーソナライズされたアドバイスを生成します。
- **機能 (Functionality)**:
    - **感情カテゴリの特定**: 音声の内容から感情（例: Happy, Calm, Sadなど）を分類します。
    - **タイトルの生成**: ジャーナルの内容を簡潔に表すタイトルを自動生成します。
    - **要約の生成**: ジャーナルの主要なポイントをまとめた要約を提供します。
    - **アドバイスの生成**: 検出された感情や内容に基づき、ユーザーへ向けた個別のアドバイスを提案します。
- **出力形式 (Output Format)**: 上記の結果は構造化されたデータ形式で提供され、アプリ内で表示・保存されます。

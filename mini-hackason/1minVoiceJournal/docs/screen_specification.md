# 📱 画面仕様 (Screen Specifications)

## 1. ホーム画面 (Home Screen)

- **目的 (Purpose)**: アプリの起動画面であり、音声ジャーナルの録音を開始する場所です。
- **主な機能 (Key Features)**:
    - 音声の録音開始と停止（最大1分間）。
    - 録音中の経過時間を表示。
    - 過去のジャーナル記録一覧画面へのナビゲーション。
- **フロー (Flow)**:
    - ユーザーが録音を完了すると、アプリは自動的に音声分析を開始し、分析結果画面に遷移します。

## 2. 分析結果画面 (Result Screen)

- **目的 (Purpose)**: AIによる音声分析の結果を確認し、ジャーナルとして保存する画面です。
- **主な機能 (Key Features)**:
    - AIが生成した「感情」「タイトル」「要約」「アドバイス」の表示。
    - 分析結果をジャーナルとして永続的に保存。
- **フロー (Flow)**:
    - ホーム画面で録音が完了した後に表示されます。
    - 「保存」ボタンをタップすると、記録が保存され、ホーム画面に戻ります。

## 3. 記録一覧画面 (Log List Screen)

- **目的 (Purpose)**: これまでに保存した全てのジャーナル記録を一覧で表示し、アクセスする画面です。
- **主な機能 (Key Features)**:
    - 保存されたジャーナルを日付、感情アイコン、タイトルでリスト表示。
    - 各ジャーナルをタップして詳細画面へ遷移。
- **フロー (Flow)**:
    - ホーム画面からいつでもアクセスできます。

## 4. 詳細画面 (Detail Screen)

- **目的 (Purpose)**: 個別のジャーナル記録の詳細内容を確認し、元の音声を再生する画面です。
- **主な機能 (Key Features)**:
    - 選択したジャーナルの全情報（感情、タイトル、日付、要約、アドバイス）を表示。
    - 記録された元の音声を再生・一時停止。
    - ジャーナル記録の削除。
- **フロー (Flow)**:
    - 記録一覧画面で特定のジャーナルを選択すると表示されます。

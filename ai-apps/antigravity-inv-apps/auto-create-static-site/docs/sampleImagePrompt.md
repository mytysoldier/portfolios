# nanobanana画像生成プロンプト集

nanobanana (画像生成AI) に渡すためのプロンプト例です。
サイト制作で使いやすい画像を生成するために、以下のカテゴリ別にコピーして使用してください。
※ 画像生成AIは英語のプロンプトの方が品質が高くなる傾向があるため、英語で記述していますが、解説を日本語で添えています。

---

## 共通のコツ
*   **アスペクト比**: `nanobanana` はアスペクト比指定（--ar）に対応していない場合があります。その場合、正方形(1:1)で生成されることが多いです。「ワイドショット」や「パノラマ」とプロンプトに含めることで構図を調整できる場合がありますが、基本は正方形をトリミングして使うことを想定してください。
*   **スタイルの統一**: 全てのプロンプトの末尾に同じスタイル呪文（例: `minimalist style, soft lighting`）を付けると、サイト全体の統一感が出ます。

---

## 1. ヒーロー画像 (Topの背景など)
**用途**: サイトの顔となるメインビジュアル。文字が乗ることを想定して「空間」がある構図がベター。

### カフェ・飲食店向け
```text
居心地の良いカフェの店内のワイドショット、窓から差し込む日差し、木製のテーブル、手前のテーブルにラテアート、背景はぼかし、温かみのある雰囲気、高解像度
```

### テック・SaaS企業向け
```text
抽象的なテクノロジー背景、青と白のデータストリーム、洗練されたモダンさ、ミニマリズム、右側にテキスト用の十分な余白、未来的な雰囲気
```

### ポートフォリオ・クリエイター向け
```text
クリエイティブな作業スペース、真上から見た少し散らかったおしゃれなデスク、スケッチブック、ペン、MacBook、コーヒー、植物、柔らかな自然光、ミニマリストスタイル
```

---

## 2. コンテンツ用画像 (商品の魅力や特徴)
**用途**: グリッドで並べるための画像。被写体が中央にはっきり映っているものが良い。

### 商品（コーヒー豆の例）
```text
木製の板の上に上品に置かれた高級焙煎コーヒー豆の袋、プロの商品写真、シャープな焦点、スタジオライティング
```

### サービス（ビジネスミーティング）
```text
モダンなオフィスで議論しているプロフェッショナルなビジネスチーム、多様なグループ、笑顔、書類を持っている、自信に満ちた雰囲気、ミディアムショット
```

### 機能イメージ（スマホアプリ）
```text
洗練されたアプリ画面を表示しているスマートフォンを持つ手の接写、ぼやけた街の背景、鮮やかな色、モダンなライフスタイル
```

---

## 3. 雰囲気・テクスチャ (背景や装飾)
**用途**: セクションの背景や、Aboutページのイメージ画像。

### 自然・オーガニック
```text
オーガニックなボタニカルパターン、緑の葉と柔らかな影、真上からの視点、自然のテクスチャ、落ち着いた背景
```

### オフィス・信頼感
```text
ぼやけたモダンな建築、ガラスのビルのファサード、青空の反射、プロフェッショナルな抽象的背景
```

---

## 4. 複数枚のバリエーション (ギャラリー用)
**用途**: ギャラリーや商品一覧のように、統一感のある画像を複数枚生成したい場合。
**コツ**: 同じプロンプトで `--count` オプションを使うか、被写体だけ少し変えてスタイルは固定します。

### カフェメニュー（統一感のある3枚）
コマンド例: `--count 3` を追加して実行
```text
白い皿に乗った美味しそうな [クロワッサン / チーズケーキ / エッグベネディクト]、カフェのテーブル、自然光、俯瞰撮影、統一されたミニマリストスタイル
```
(解説: []の中身を変えて複数回生成するか、--countを使ってバリエーションを出します。)

### ポートフォリオ作品（グリッド表示用）
```text
モダンなポスターデザイン、[赤 / 青 / 黄] を基調とした抽象的な幾何学模様、スタジオ照明、高品質なプリントテクスチャ
```

---

## 5. スタイル指定の定型文 (Suffix)
プロンプトの最後につけるとクオリティが上がる呪文です。

*   **写真リアル**: `フォトリアル、8k、高詳細、35mmレンズで撮影`
*   **イラスト・3D**: `3Dレンダリング、Blenderスタイル、ソフトパステルカラー、キュートで丸みのある`
*   **ミニマリスト**: `ミニマリスト、きれいな線、白背景、ソフトな影`

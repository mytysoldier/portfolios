# 推奨プロンプト集 (Recommended Prompts)

以下は、このツールで高品質なサイトを生成するためのプロンプトの例です。コピー＆ペーストして、カッコ内の内容を自分のプロジェクトに合わせて調整してください。日本語で指示しても問題ありません。

## 1. クリエイティブ・ポートフォリオ
**特徴**: 視覚重視、シンプル、モダンなデザイン。
```text
[グラフィックデザイナー / 写真家] 向けの、ミニマリストでモダンなポートフォリオサイトを作ってください。
ページ構成: ホーム、ポートフォリオ、お問い合わせ。
スタイル: ダークモード、余白を効果的に使う、プレミアムで洗練された雰囲気。
コンテンツ:
- ホーム: 「瞬間を切り取り、アートを創造する」というキャッチコピーを目立たせるヒーローセクション。
- ポートフォリオ: 私の最高傑作をグリッドレイアウトで表示（提供された画像を使用）。
- お問い合わせ: メールとSNSリンクを含むシンプルなフォーム。
トーン: プロフェッショナル、アーティスティック、自信に満ちた。
```

## 2. 企業のコーポレートサイト
**特徴**: 信頼性、情報整理、明確な構成。
```text
[SaaSスタートアップ / コンサルティング会社] 向けのプロフェッショナルなコーポレートサイトを構築してください。
ページ構成: ホーム、事業内容、会社概要、お問い合わせ。
スタイル: 清潔感、信頼感（青と白の配色）、レスポンシブなレイアウト。
コンテンツ:
- ホーム: 「未来を革新する」というヒーローセクションと、その後に「選ばれる理由」セクション。
- 事業内容: [サービスA]、[サービスB]、[サービスC] のそれぞれのセクション。
- 会社概要: ミッションステートメントとチーム紹介。
トーン: 信頼できる、革新的、顧客第一。
```

## 3. 商品/サービスのランディングページ (LP)
**特徴**: コンバージョン重視、強い訴求。
```text
[新しいコーヒーブランド] のための、コンバージョン率の高いランディングページ(LP)を作成してください。
ページ構成: ホーム(LP)、商品詳細(Shop)。
スタイル: 暖かみがある、魅力的、オーガニックな色使い（茶色、緑、クリーム色）。
コンテンツ:
- ヒーロー: 「至高の一杯を体験しよう」というコピーと、「今すぐ購入」という大きなCTAボタン。
- 特徴: 「倫理的な調達」、「プレミアムロースト」、「エコフレンドリーなパッケージ」。
- お客様の声: 「今まで飲んだ中で最高のコーヒー！」 - 山田花子
トーン: フレンドリー、情熱的、親しみやすい。
UIルール: 「今すぐ購入」ボタンを目立たせ、Shopページへリンクさせること。
```

## ヒント
- **日本語でOK**: AIは日本語を理解しますので、詳細なニュアンスも日本語で指示して大丈夫です。
- **具体的に**: 「かっこいいサイト」より「黒を基調とした、アニメーション多めのテック系サイト」のように具体的に書くと精度が上がります。

## 4. 画像素材がある場合 (With Images)
**特徴**: 既存のデザインカンプや素材画像を読み込ませて、それを再現・活用する場合。
**前提**: `--image` オプションで画像を読み込んでいること。
```text
読み込ませた画像を最大限に活用して、[アパレルブランド] のECサイトを作成してください。
ページ構成: Top, Items, BrandStory.
スタイル: 画像のデザインテイスト（色味、フォント感）に忠実に合わせてください。
コンテンツ:
- メインビジュアル: 提供した画像素材を使用。
- アイテム一覧: 読み込んだ商品画像をギャラリー形式で並べる。
- 補足: 画像からブランドの雰囲気（高級感、ストリート系など）を読み取り、それに適したコピーライティングを行ってください。
注意点: 画像は決してリンク化せず、静的な表示のみに留めてください。
```

## 5. カフェ・レストラン (Cafe / Restaurant)
**特徴**: 雰囲気重視、メニュー紹介、アクセス情報。
```text
[レトロな隠れ家カフェ] のための、居心地の良さが伝わるウェブサイトを作成してください。
ページ構成: Home, Menu, Access.
スタイル: ベージュや木目調をベースにした、落ち着いた温かみのあるデザイン。フォントは明朝体や手書き風を検討。
コンテンツ:
- Home: 店内の雰囲気がわかる大きな写真と、「都会の喧騒を忘れる場所」というキャッチコピー。
- Menu: おすすめのコーヒーと自家製ケーキを、写真付きで美味しそうに紹介。価格も記載。
- Access: 地図（プレースホルダー）、営業時間、最寄り駅からの道のり。
トーン: 穏やか、丁寧、歓迎ムード。
```

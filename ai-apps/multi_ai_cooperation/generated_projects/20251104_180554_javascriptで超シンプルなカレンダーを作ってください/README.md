# 超シンプルカレンダー

JavaScriptで作成した超シンプルなカレンダーアプリケーションです。

## 機能

### 必須機能
- ✅ 現在月のカレンダー表示
- ✅ 曜日表示（日〜土）
- ✅ 現在日のハイライト表示

### 実装済みオプション機能
- ✅ 月移動機能（前月/次月ボタン）
- ✅ 日付クリック時のアラート表示
- ✅ キーボード操作（←→キーで月移動）
- ✅ 日曜日・土曜日の色分け表示
- ✅ レスポンシブデザイン

## ファイル構成

```
calendar/
├── index.html    # HTMLファイル（カレンダーの骨格）
├── style.css     # CSSファイル（スタイル定義）
├── script.js     # JavaScriptファイル（カレンダーロジック）
└── README.md     # このファイル
```

## 実行方法

1. すべてのファイルを同じディレクトリに配置します
2. `index.html` をWebブラウザで開きます
3. カレンダーが表示されます

### 対応ブラウザ
- Chrome（最新版）
- Firefox（最新版）
- Safari（最新版）
- Edge（最新版）

## 使い方

- **月の移動**: 左右の矢印ボタンをクリック、またはキーボードの←→キーを使用
- **日付選択**: 日付をクリックするとアラートが表示されます
- **今日の日付**: 紫色でハイライト表示されます

## カスタマイズ方法

### 色の変更
`style.css` ファイルの以下の部分を編集してください：

```css
/* メインカラー */
.nav-button {
    background: #667eea;  /* ボタンの色 */
}

.day.today {
    background: #667eea;  /* 今日の日付の背景色 */
}

/* 背景のグラデーション */
body {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### 日付クリック時の動作変更
`script.js` の `createDayElement` 関数内のイベントリスナー部分を編集してください：

```javascript
dayElement.addEventListener('click', () => {
    // ここに任意の処理を記述
    alert(`${year}年${month}月${date}日がクリックされました`);
});
```

## 技術仕様

- **言語**: JavaScript (ES6+)
- **マークアップ**: HTML5
- **スタイル**: CSS3
- **外部依存**: なし（Pure JavaScript実装）

## 主な実装ポイント

1. **日付計算**: JavaScriptの`Date`オブジェクトを使用
2. **DOM操作**: グリッドレイアウトで動的に日付を生成
3. **今日の日付ハイライト**: 現在日と比較して自動的にクラスを付与
4. **月またぎ処理**: 前月・次月の日付も表示してカレンダーを完全に埋める

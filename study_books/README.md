# 学習教科書集

このリポジトリには、プログラミング言語やフレームワークに関する学習教科書が含まれています。

各教材は、理論と実践を組み合わせたステップバイステップ形式で、段階的に学習できるよう設計されています。

## ⚠️ 重要な注意事項

**これらの教材はAIによって生成されています。** そのため、誤った内容や古い情報が含まれる可能性があります。学習の際は以下の点にご注意ください：

- 公式ドキュメントや最新の情報源と照らし合わせて確認してください
- コード例は実際に動作することを確認してから使用してください
- 誤りを見つけた場合は、フィードバックをお願いします
- 実務で使用する場合は、必ず公式ドキュメントを参照してください

## 📚 利用可能な教材

### Swift

#### [TCA (The Composable Architecture)](./swift/tca/)
SwiftUIアプリケーションのためのアーキテクチャパターン「The Composable Architecture」の学習教材です。

- **[1時間短縮版](./swift/tca/1hour/)**: TCAの基本を素早く理解したい方向け
- **[3時間みっちり版](./swift/tca/3hours/)**: TCAを深く理解し、実践的なアプリケーションを構築できるようになるための詳細版

## 🎯 教材の特徴

- **Phase分け**: 各教材は複数のPhaseに分かれており、段階的に学習できます
- **理論と実践**: 各Phaseには理論の説明と実践的なコード例が含まれています
- **練習問題**: 各Phaseの最後には理解を深めるための練習問題があります
- **独立した構成**: 各教材は独立して学習できるようになっています

## 📖 学習の進め方

1. 興味のある言語・フレームワークの教材を選択
2. 各教材のREADMEを読んで、学習目標と構成を確認
3. Phaseを順番に進めながら、実際にコードを書きながら学習
4. 練習問題に取り組んで理解を深める

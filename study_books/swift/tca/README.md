# TCA (The Composable Architecture) 学習教材

このディレクトリには、SwiftのTCA（The Composable Architecture）を学習するための教材が含まれています。

## 教材の種類

### [1時間短縮版](./1hour/)
TCAの基本を素早く理解したい方向けの短縮版です。
- **対象**: TCAを初めて学ぶ方、時間が限られている方
- **内容**: 基本的な概念と簡単な実践例
- **所要時間**: 約1時間
- **詳細**: [1時間版のREADME](./1hour/README.md)を参照

### [3時間みっちり版](./3hours/)
TCAを深く理解し、実践的なアプリケーションを構築できるようになるための詳細版です。
- **対象**: TCAをしっかりと理解し、実務で使えるようになりたい方
- **内容**: 理論から実践、テストまで包括的に学習
- **所要時間**: 約3-4時間（理解度により変動）
- **詳細**: [3時間版のREADME](./3hours/README.md)を参照

## 学習の進め方

1. まずは[1時間版](./1hour/)で基本を掴む
2. より深く学びたい場合は[3時間版](./3hours/)に進む
3. 各Phaseの実践例を実際にコードを書きながら進める
4. 各Phaseの最後にある練習問題に取り組む

## 各教材について

各教材は独立して学習できるようになっています。各フォルダ内のREADMEを参照してください。
